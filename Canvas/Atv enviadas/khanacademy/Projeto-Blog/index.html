Principais linguagens


Existem muitas formas de classificar as linguagens de programação, podem ser classificadas quanto ao modo declarativo e imperativo, estrutura de tipos, grau de abstração, entre outras definições. É importante pesquisar um pouco para que serve cada área, para que a partir disso conseguir decidir qual a melhor linguagem para aprender. A finalidade de aprender um pouco sobre cada área é por exemplo: Conseguir identificar qual a melhor linguagem para desenvolver determinada aplicação, visto que um projeto pode ser melhor implementado com uma linguagem que com outra.


Tipos de linguagens

Front-end, back-end e mobile


Uma das áreas da programação é o front-end que trabalha com o desenvolvimento web e lida bastante com a parte "visual" do projeto, onde o profissional precisa criar algo que seja fácil de ser utilizado, que seja acessivél para todos os usuários e também precisa ser um projeto responsivo, ou seja, que seja funcional e tenha boa usabilidade em qualquer tipo de plataforma.
O profissional back-end é o que trabalha "por trás" do que o usuário está vendo na sua tela, ele desenvolve o processo lógico de instruções e comandos que faz com que as máquinas façam o que queremos.
O programador mobile é o profissional que desenvolve softwares e recursos para dispositivos móveis.
